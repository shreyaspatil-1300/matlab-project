Maxima and Minima – Complete Theory

Introduction

Maxima and Minima is an important topic in Differential Calculus that deals with finding the greatest and smallest values of a function. These concepts are widely used in mathematics, physics, engineering, economics, optimization problems, and real-life applications.

  • Maximum Value → The highest value attained by a function.
  • Minimum Value → The lowest value attained by a function.

The process of finding maxima and minima is called Optimization.

1. Basic Definitions

Consider a function y = f(x).

1.1 Local Maximum

f(x) has a local maximum at x=a if f(a) > f(x) for all values of x near a. The graph rises before the point and falls after.

1.2 Local Minimum

f(x) has a local minimum at x=a if f(a) < f(x) for all nearby x. The graph falls before and rises after.

1.3 Absolute (Global) Maximum

The largest value of the function over its entire domain.

1.4 Absolute (Global) Minimum

The smallest value of the function over its entire domain.

2. Critical Points

x = a is a critical point if f'(a)=0 or f'(a) does not exist. Critical points are possible locations of maxima or minima.

3. Necessary Condition for Maxima/Minima

If a differentiable function f(x) has a maximum or minimum at x=a, then f'(a)=0. The tangent is horizontal.

4. Increasing and Decreasing Functions

Increasing: f'(x) > 0 (positive slope). Decreasing: f'(x) < 0 (negative slope).

5. First Derivative Test

Procedure: Find f'(x), solve f'(x)=0, check sign of f'(x) on both sides.

Sign change of f'(x)Nature
+ → -Local Maximum
- → +Local Minimum
No changeNeither

6. Second Derivative Test

Find f''(x). At critical point a: if f''(a)>0 → Minimum; f''(a)<0 → Maximum; f''(a)=0 → Test fails.

7. Concavity

f''(x) > 0 → concave upward (cup shape). f''(x) < 0 → concave downward (cap shape).

8. Point of Inflection

Where concavity changes: f''(x)=0 and sign changes.

9. Stationary Points

Points where f'(x)=0: stationary maximum, stationary minimum, stationary point of inflection.

10. Non-Differentiable Points

Extrema may occur where derivative does not exist, e.g., f(x)=|x| at x=0 (minimum).

11. Absolute Extrema on Closed Intervals [a,b]

Find critical points inside, evaluate f at critical points and endpoints, compare. Largest = absolute max, smallest = absolute min.

12. Geometrical Interpretation

At maxima/minima, tangent is horizontal: dy/dx = 0.

13. Maxima and Minima for Functions of Two Variables

For f(x,y), find partial derivatives f_x = 0, f_y = 0. Compute D = f_xx f_yy - (f_xy)².

ConditionNature
D>0, f_xx>0Minimum
D>0, f_xx<0Maximum
D<0Saddle Point
D=0Inconclusive

14. Optimization Problems

Examples: max profit, min cost, max area, min distance. Procedure: draw figure, define variables, form equation, reduce to one variable, differentiate, set derivative zero, apply tests, state answer.

15. Important Standard Results

Quadratic f(x)=ax²+bx+c: vertex at x=-b/(2a). a>0 → minimum, a<0 → maximum. Polynomials: even degree generally have both max/min; odd degree may not.

16. Important Formulas

  • Necessary condition: f'(x)=0
  • First derivative test: +→- max; -→+ min
  • Second derivative test: f''(x)>0 min, f''(x)<0 max

17. Common Mistakes

  • Forgetting to check critical points
  • Ignoring endpoints on closed intervals
  • Not checking sign changes
  • Forgetting points where derivative does not exist

18. Summary of Core Concepts

First derivative tells increase/decrease; second derivative tells curvature. Slope zero → possible extrema. Increasing to decreasing → maximum. Decreasing to increasing → minimum.

19. Step-by-Step Methods

First Derivative Test: find f'(x), solve f'(x)=0, check sign intervals, classify.

Second Derivative Test: find f'(x)=0, compute f''(x), substitute critical points, classify.

20. Example Problem

f(x)=x³-3x²+2. f'(x)=3x²-6x=3x(x-2) → critical x=0,2. f''(x)=6x-6. f''(0)=-6<0 → maximum; f''(2)=6>0 → minimum. f(0)=2, f(2)=-2. Answer: Max value 2 at x=0, Min value -2 at x=2.

22. Conclusion

The theory of maxima and minima is fundamentally based on derivatives and function behavior. The first derivative determines increase/decrease, the second derivative gives curvature. These concepts enable solving optimization problems and analyzing functions in mathematics and applied sciences.
