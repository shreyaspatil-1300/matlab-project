Maxima, Minima & Saddle Points

Master the art of identifying peaks, valleys, and saddle points in functions of two variables.

Local / Global Maximum

A point (a,b) is a local maximum if f(x,y) ≤ f(a,b) near that point. If it holds over entire domain → Global maximum.
Mathematically: First derivatives zero and second derivative test D>0, r<0.

Local / Global Minimum

Analogous, with f(x,y) ≥ f(a,b) around point → Minimum. Condition: D>0, r>0.

Saddle Point

Critical point which is neither max nor min — surface curves up in one direction and down in another. Condition: D < 0.

Second Derivative Test

Let r = fxx, s = fxy, t = fyy, and D = r·t - s².
➜ D>0 & r>0 → min ; D>0 & r<0 → max ; D<0 → saddle ; D=0 → inconclusive.

How to identify: Find points where ∂f/∂x = 0 and ∂f/∂y = 0 (Critical Points). Then evaluate second derivatives at those points. Use the computed D value to classify.
