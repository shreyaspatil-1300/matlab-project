Maxima, Minima & Saddle Points

Multivariable calculus: Learn to classify critical points using second derivative test.

Maximum

Local max: f(x,y) ≤ f(a,b) nearby. Condition: D>0 and f_xx<0.

Minimum

Local min: f(x,y) ≥ f(a,b). Condition: D>0 and f_xx>0.

Saddle Point

Critical point with D<0 → neither max nor min.

Second Derivative Test

r = f_xx, s = f_xy, t = f_yy, D = rt - s².
D>0 & r>0 → min ; D>0 & r<0 → max ; D<0 → saddle ; D=0 → inconclusive.
